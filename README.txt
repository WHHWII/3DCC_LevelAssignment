Whitman Wilson Level Renderer Project 
3DCC 2023


Controls:
Left mouse to move camera
WASD for lateral movement
Shift for down
space for up
F1 to load a different level
1 to load default gShader
2 to load Primative explosion gShader


Core features: 
all

Key Features:
Level loading (press f1, select gametext. It loads models from a model folder containing all models needed for the provided levels) 
Instanced rendering.


Advanced Features 
Geometry shader (press 1 for default, 2 for prim explosion)
